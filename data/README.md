# Data Directory

This directory should contain the following CSV files from the Olist Brazilian E-Commerce dataset:

## Required Files:
- olist_orders_dataset.csv
- olist_order_items_dataset.csv
- olist_order_reviews_dataset.csv
- olist_products_dataset.csv
- product_category_name_translation.csv
- olist_customers_dataset.csv

## Download Instructions:
1. Visit: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
2. Download the dataset
3. Extract all CSV files into this `data/` folder

## File Size Warning:
These files are NOT included in this repository due to GitHub size limits.
Total size of required files is approximately 57MB.
